React app that serves static HTML using [react-snap](https://www.npmjs.com/package/react-snap).

## Build

* Install [Node](https://nodejs.org/)
* Install dependencies
    ```sh
    npm i
    ```
* Build
    ```sh
    npm run build
    ```

## Local development

```sh
npm run dev
```
