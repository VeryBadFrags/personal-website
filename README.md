## Build

- Install [Node](https://nodejs.org/)
- Install dependencies

  ```sh
  pnpm i
  ```

- Build

  ```sh
  pnpm run build
  ```

## Local development

```sh
make dev
```

### Font Awesome

Create a `.env` file and add a link to your own Font Awesome Kit:

```.env
FONTAWESOME_KIT_URL=https://kit.fontawesome.com/your-kit-id.js
```
