A static website built with [Astro](https://astro.build/).

## Build

- Install [Node](https://nodejs.org/)

### With [pnpm](https://pnpm.io)

```sh
make build
```

### With npm

```sh
npm i
npm run build
```

## Local development

```sh
make dev
```
